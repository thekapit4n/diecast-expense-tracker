## What Changed

### 1) New Catalog Page — `app/catalog/`

A brand new mobile-first catalog page was built from scratch.

**Files created:**
- `app/catalog/page.tsx` — server component that fetches all data from Supabase
- `app/catalog/layout.tsx` — standalone full-screen layout (no admin sidebar)
- `app/catalog/components/CatalogClient.tsx` — main interactive client component
- `app/catalog/components/DiecastCard.tsx` — individual model card
- `app/catalog/components/BrandTabs.tsx` — horizontal scrollable brand filter
- `app/catalog/components/CardDetailSheet.tsx` — bottom drawer with model detail + purchase history
- `app/catalog/components/FilterSortSheet.tsx` — filter and sort bottom sheet

**Features:**
- Dark automotive theme (`#0B0B0C` background, `#16181D` cards, `#FF3B30` accent red)
- 2-column mobile grid / 3-column tablet / 4–5 column desktop
- Sticky header with model count, brand count, search icon, filter icon, and burger menu
- Horizontal scrollable brand tabs — defaults to Mini GT on load
- Only shows brands with actual collection data (dynamic, auto-updates as you add models)
- Only shows `Diecast` type brands — Diorama and Accessory brands are excluded
- Mini GT list ordered by series number (MGT00001, MGT00002…) by default
- Search by model name, item number, brand, or scale
- Filter sheet: brand multi-select, scale multi-select, sort options
- Card overlays: `Owned` badge (blue), `Chase` badge (red), `Case` badge (amber), scale label
- Tap any card → opens bottom drawer showing:
  - Image carousel (swipe left/right with dot indicators)
  - Model name, item number, brand, scale badges
  - Purchase history sorted by highest price first
  - Each purchase row shows: shop name, platform, payment date, unit price, quantity
  - Total price shown when quantity > 1
  - Chase unit label per purchase row
- Load more pagination (20 items per batch)

### 2) Public Image Proxy API — `app/api/catalog/image/[series]/[fileName]/route.ts`

A new image serving endpoint that does **not** require a user session.

- Mirrors the existing `/api/mini-gt/image/` route but without the auth check
- Downloads from Supabase Storage bucket using the anon key
- Falls back to local `public/images/mini-gt/` files if storage fails
- Sets `Cache-Control: public, max-age=86400` (24hr browser cache)
- Remark-stored image URLs (previously pointing to `/api/mini-gt/image/`) are automatically rewritten to `/api/catalog/image/` so they work without login

### 3) Supabase RLS Migration — `supabase/migrations/20260517000000_add_public_anon_select_policies.sql`

Added SELECT-only policies for the `anon` role on four tables:

- `tbl_master_brand`
- `tbl_collection`
- `tbl_collection_detail`
- `tbl_purchase`

The existing `authenticated` ALL policies (for insert/update/delete) are untouched.
These policies are required for the catalog page to read data.

> **Note:** Run `supabase db push` to apply this migration to the remote database.

### 1) New Color Theme

The catalog no longer uses flat black (`#0B0B0C`). It now uses a dark teal palette inspired by the chameleon diecast paint:

| Role | Color |
|------|-------|
| Page background | `#0b1822` |
| Cards / surfaces | `#122030` |
| Borders | `#1d3344` |
| Primary accent | `#3c647b` |
| Owned / positive | `#669a62` |
| Model title (card + modal) | `#BFE9FF` |

**Files updated:** `layout.tsx`, `CatalogClient.tsx`, `DiecastCard.tsx`, `BrandTabs.tsx`, `FilterSortSheet.tsx`, `CardDetailSheet.tsx`

### 2) Model Detail Drawer (Modal)

### 3) Sorting Fix

**Problem:** Non–Mini GT brands (e.g. Tarmac Works) looked random because old sort options (`Newest First`) did nothing and left items in database insert order.

**Solution:**
- Data loads from Postgres ordered by `name` ASC
- New `sortCatalogItems()` always sorts after filtering
- **Mini GT** → sort by full `item_no` (MGT00001, MGT00002, …)
- **All other brands** → always sort by model name A–Z
- Removed broken "Newest First" / "Oldest First" options from the filter sheet
- Changing brand tab resets sort to the correct default

### 4) Burger Menu Styling

- Close button: frosted glass look, icon `#94A3B8`, hover `#B6FFF2`
- Divider under "Diecast Tracker": `rgba(125,211,252,0.06)`

- Active page (Catalog): text `#99F6E4`, icon `#5EEAD4` with soft glow
